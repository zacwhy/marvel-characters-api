# Marvel Characters API


## Getting Started

Create `config.json`
```sh
$ cp config.json.example config.json
```

Modify the configurations in `config.json`

Install dependencies
```sh
$ npm install
```

Start API server
```sh
$ npm start
```


## Development

To run test
```sh
$ npm test
```
